1. Show last log records: journalctl -b -1
2. Start service automatically: systemctl enable <service_name>.service
3. List all the packages with size specified:
    pacman -Qi | egrep '^(Name|Installed)' | cut -f2 -d':' | tr '\nK' ' \n' | sort -nrk 2 | less

4. Set power profile
    sudo cpupower frequency-set -g powersave
    sudo cpupower frequency-set -g conservative
    sudo cpupower frequency-set -g performance
